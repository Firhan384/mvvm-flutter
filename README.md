# mvvm_flutter

Boiler Plate Flutter with implementation MVVM(Model View Model-View)

## Structure Folder

- lib
  - src
  - pages
    - to-do-list
      - model
      - view
      - view_model
  - routes
  - utils
  - providers

## Penerapan MVVM

- view : digunakan untuk hanya berfokus kepada ui
- model : data yang akan direpresentasikan
- view model : logika bisnis
